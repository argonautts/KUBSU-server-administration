# Study task docker, docker-compose, Makefile

### Условие задачи

Написать приложение, которое будет ходить в БД Postgres на этой машине с кредами kubsu/kubsu к БД kubsu и выводить все записи из таблицы users. В проекте должны быть:
- Taskfile/Makefile для быстрого запуска всех команд
- тест на этот единственный эндпоинт
- docker-compose для удобства локальной разработки

### Структура проекта

- Папка Gunicorn - wsgi-сервер, который выводит записи из таблицы на страницу. Добавлен тест, который проверяет доступна ли страница по адресу localhost:8000
- Папка ConsoleOutput - выводим записи из таблицы в консоль. Добавлен тест, который сверяет записи из таблицы
